== Description ==
Let users  upload documents to your Dropbox folder. Author url - http://webania.net


== Installation ==

Simple:

1. Descompress the file
2. Upload the directory `dbouploader/` to the `wp-content/plugins/` directory.
3. Activate the plugin
4. Go to Plugin editor, open config.php, set upload folder, YOUR DROPBOX EMAIL  and YOUR DROPBOX PASSWORD and replace them with your Dropbox login data. 
Save document. Shortcode is [dbouploader]. Paste this shortcode anywhere you want in your posts&pages. Find more information here: http://wp.tutsplus.com/tutorials/simple-plugin-which-lets-users-upload-files-into-your-dropbox-folder/
