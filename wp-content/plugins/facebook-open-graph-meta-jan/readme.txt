Facebook Open Graph Meta is a wordpress plugin that will help you to add facebook open graph protocol meta data in your blog.

== Description ==

Facebook Open Graph Meta is a wordpress plugin that will help you to add open graph protocol meta data in your blog. When your visitors share or like your blog post or page the shared post will show on his facebook wall in a nice way.
So that his friends get inspire to checkout the shared post.

== Installation ==

1. Upload facebook_open_graph_meta/ into wp-content/plugins/
2. Activate the plugin via the 'Plugins' menu.
3. Go to the 'Facebook open graph' option under settings.
4. Follow the tutorial on that page to create facebook application
5. At the last step you've to update your theme's header file
